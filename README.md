# Training-Banking-JSVanilha-CSS

Exemplo criado para Vagas de emprego de um Layout CSS e armazenamento LocalStorage 

## Competências

- **CSS** - Criação de arquivo CSS de página completa 
- **LocalStorage** - Utilização de armazenamento em Cache do navegador 
- **Vanilha JS** - Utilização de script de página sem utilização de Framework
